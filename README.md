# Trajectory
Trajectory Code for collaborative robot B1 from Universal Robots Mexiko. 

There was a collaboration between the students of the Tecnológico de Monterrey and Universal Studios to build a automated and closed-loop assembly process for 3D-printed supportive springs. It part of the final challenge of Cyber Physical Systems I with the big partner OMRON Robots. 

The objective was to program and equip the robot B1 well enough to drive selfby for pick and place the prototype fixture to the goal rack. 

There were used ultrasonic and infrared camera sensors to detect obstacles for bypass. 
The whole code was written with Python 3 in ROS I. 
For right orientation in the laboratory there was used odometry with starting point as 0/0/0.
